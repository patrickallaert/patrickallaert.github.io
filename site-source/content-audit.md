# Content Audit

This file separates the public snapshot into the working set for the new EN-only Conexao site.

## Recommendation

- Keep the raw snapshot intact in `site-source/html`.
- Build only from `core-en` plus `seasonal-reference` and `legal-reference`.
- Ignore `fr-archive`, `archive`, and probably `blog-reference` for the first launch.
- Replace recurring trimester pages with one reusable class-cycle model: September, January, April.
- On the live site, show the current trimester and, when relevant, the next trimester once registration opens.
